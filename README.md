# My Personal Website

One already exists at [jarmos.netlify.app](https://jarmos.netlify.app) which was created using [Pelican](https://getpelican.com). While it served a primary purpose of hosting & delivering content for me, it's hard to customize according to my needs. So I choose to go with Hugo & the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

It's easier to customize & use but how efficient is it remains to be seen.
